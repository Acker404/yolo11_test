Qt YOLO 應用程式使用說明
========================

檔案說明：
- debug_start.bat    : 除錯模式啟動
- check_system.bat   : 系統環境檢查

安裝步驟：
3. 如果環境正常，執行 start.bat 啟動程式
4. 如果程式無法啟動，執行 debug_start.bat 除錯

常見問題：
   解決方法：安裝 Microsoft Visual C++ 2019-2022 Redistributable (x64)

2. DirectX 版本過舊
   解決方法：更新 DirectX 到最新版本

3. 防毒軟體阻擋
   解決方法：將程式資料夾加入防毒軟體白名單

4. 權限不足
   解決方法：以管理員身分執行

技術支援：請提供 debug_start.bat 的輸出結果
